## Short Code Git
1. 
	```
	git clone URL 
	```
1. 
	```
	git status 
	```
1. 
	```
	git add . 
	```
1. 
	```
	git config --global user.name       "/* Your GitHub UserName */" 
	```
1. 
	```
	git config --global user.email      "/* Your GitHub Email */" 
	```
1. 
	```
	git config -l 
	```
1. 
	```
	git config --global credential.helper cache 
	```
1. 
	```
	git pull 
	```
1. 
	```
	git commit -m "Msg" 
	```
1. 
	```
	git push 
	```
